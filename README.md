# おしんこ屋さん v9.0

ドット絵レトロ風の漬物屋経営シミュレーションゲーム。
明治初期の架空の地方都市・椿町を舞台に、三人の看板娘と共に小さな漬物屋を切り盛りします。

## 遊び方

朝の市場で野菜を仕入れ → 樽で漬け込み → **店頭に並べる** → 値段を決めて → お客に売る。
さらに料理レシピで加工品も作って販売できます。

## 加工品システム(v9で大幅刷新)

- 樽は加工品の素材になるけど**消費されない**(母樽として何度でも使える)
- 加工品在庫の上限は**初期8個**、増設で最大24個まで拡張可能(¥3000〜¥15000)
- 加工品も「店頭」と「倉庫」を切替可能(店頭の品だけ売れる)
- 加工品の種類は**12種類**:
  - 浅漬けから:しば漬け和え/浅漬けサラダ/冷やしそばの薬味(夏限定)
  - 食べごろから:おしんこ巻き/おにぎりの友/お弁当用詰合せ
  - 古漬けから:お茶漬けの素/お酒の肴/高級茶漬けセット
  - 看板娘特製:お梅の田舎漬け詰め/お菊銘柄/お雪の小鉢仕立て

## 物語

椿町の片隅にある小さな漬物屋。素性の知れない店主・黒井さんは、ほとんど店に出てこない。
店を任されたあなたは、三人の看板娘たちと共に、店の物語を紡いでいきます。

- **お梅(13歳・第1週から)** — 桜沢村から奉公に出された素朴な働き者
- **お菊(17歳・第2週から)** — 没落した呉服商の娘、商才で身を立てたい
- **お雪(20歳・第5週から)** — 深川の元芸者、自分の足で立つ決意

- 第4週: ¥12,000 / 第5週以降: ¥15,000 + (章-5)×¥2,500

## エンディング

- 🏯 **名店ルート** — 総売上¥30,000以上
- 🌸 **円満ルート** — 評判75以上
- 📖 **物語ルート** — ストーリーイベントを2つ以上見て達成
- 🍂 **静かな別れ** — それ以外

1年(28日)で大エンディング、もしくはヘッダーの🌅引退ボタンから任意で。

## 機能

- 12種類の漬物 + 12種類の加工品
- 4季節7日制(1週間=1章、4週=1年)
- 3人の看板娘とそれぞれの個別エピソード
- 樽の店頭/熟成切替・加工品の店頭/倉庫切替
- 味マッチング(辛口・酸味・甘味・塩辛・さっぱり)
- 全18種の漬物レシピ + 12種の加工品レシピ
- 樽購入(最大12個)・加工品棚購入(最大24個)
- localStorage自動セーブ
- マルチエンディング+任意引退+1年経過エンディング
- 「まだ続ける」で延長可能
- 歴代ランキング(TOP10)

## 技術

- HTML + CSS + JavaScript(単一ファイル)
- ドット絵スプライトはCanvas APIで描画
- セーブ・ランキングはブラウザのlocalStorage
- 外部ライブラリなし
