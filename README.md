# おしんこ屋さん v10.0

ドット絵レトロ風の漬物屋経営シミュレーションゲーム。
明治初期の架空の地方都市・椿町を舞台に、三人の看板娘と共に小さな漬物屋を切り盛りします。

## v10.0の修正点

- **画像表示バグを修正**: 看板娘のイラストが正しく表示されるようになりました
  (これまでフォールバック処理のエスケープミスで、画像があってもドット絵が出ていました)
- **店主イラスト対応**: 店主紹介画面(🔍ボタン)で店主のイラストを表示

## 画像ファイル

`images` フォルダに以下を配置(全て小文字・拡張子.png):

```
images/oume_dot.png       お梅(ドット絵・市場の小アイコン用)
images/oume_illust.png    お梅(イラスト・登場画面用)
images/okiku_dot.png      お菊(ドット絵)
images/okiku_illust.png   お菊(イラスト)
images/oyuki_dot.png      お雪(ドット絵)
images/oyuki_illust.png   お雪(イラスト)
images/owner_illust.png   店主(イラスト・店主紹介画面用)※任意
```

画像が無い場合は自動的にコード内蔵のドット絵にフォールバックします。

## 遊び方

朝の市場で野菜を仕入れ → 樽で漬け込み → 店頭に並べる → 値段を決めて → お客に売る。
料理レシピで加工品も作って販売できます。

## 物語

- **お梅(13歳・第1週から)** — 桜沢村から奉公に出された素朴な働き者
- **お菊(17歳・第2週から)** — 没落した呉服商の娘、商才で身を立てたい
- **お雪(20歳・第5週から)** — 深川の元芸者、自分の足で立つ決意
- **店主・黒井** — 元情報屋の謎めいた店主

## エンディング

- 🏯 名店ルート(総売上¥30,000以上)
- 🌸 円満ルート(評判75以上)
- 📖 物語ルート(ストーリーイベント2つ以上)
- 🍂 静かな別れ(それ以外)

1年(28日)で大エンディング、もしくはヘッダーの🌅引退ボタンから任意で。

## 機能

- 12種類の漬物 + 12種類の加工品
- 4季節7日制(1週間=1章、4週=1年)
- 3人の看板娘とそれぞれの個別エピソード
- 樽の店頭/熟成切替・加工品の店頭/倉庫切替
- 味マッチング(辛口・酸味・甘味・塩辛・さっぱり)
- 全18種の漬物レシピ + 12種の加工品レシピ
- 樽購入(最大12個)・加工品棚購入(最大24個)
- localStorage自動セーブ
- マルチエンディング+任意引退+1年経過エンディング
- 歴代ランキング(TOP10)

## 技術

- HTML + CSS + JavaScript(単一ファイル)
- ドット絵スプライトはCanvas APIで描画
- セーブ・ランキングはブラウザのlocalStorage
- 外部ライブラリなし
