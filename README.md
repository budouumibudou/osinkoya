## 全国ランキングの設定方法(任意)

オンラインランキングを使うには、無料のFirebaseを設定します。設定しなくてもローカル(自分の端末)ランキングは動きます。

1. https://console.firebase.google.com/ にアクセス(Googleアカウントでログイン)
2. 「プロジェクトを追加」で適当な名前のプロジェクトを作成
3. 左メニュー「構築」→「Realtime Database」→「データベースを作成」
   - ロケーションは「us-central1」等でOK
   - セキュリティルールは「**テストモードで開始**」を選択
4. 作成後、画面上部に表示される **databaseのURL** をコピー
   (例: `https://oshinkoya-xxxx-default-rtdb.firebaseio.com`)
5. `index.html` をエディタで開き、次の行を探す:
   ```javascript
   const FIREBASE_DB_URL = '';
   ```
6. 空文字の部分にコピーしたURLを貼る:
   ```javascript
   const FIREBASE_DB_URL = 'https://oshinkoya-xxxx-default-rtdb.firebaseio.com';
   ```
7. GitHubの index.html を更新すれば、🏆ボタンに「全国ランキング」タブが出ます

注意: テストモードは30日で書き込みが止まります。続ける場合はFirebaseのルールでread/writeをtrueにしてください(個人の遊び用なら十分)。

## 遊び方

朝の市場で野菜を仕入れ → 樽で漬け込み → 店頭に並べる → 値段を決めて → お客に売る。
料理レシピで加工品(キムチ・べったら漬け等)も作って販売できます。

## 物語

- **お梅(13歳・第1週から)** — 桜沢村から奉公に出された素朴な働き者
- **お菊(17歳・第2週から)** — 没落した呉服商の娘、商才で身を立てたい
- **お雪(20歳・第5週から)** — 深川の元芸者、自分の足で立つ決意
- **店主・黒井** — 元情報屋の謎めいた店主

## 味の相性

お客にはそれぞれ好みの味があります:
- おばあちゃん → 酸味(古漬け)
- サラリーマン → 辛口(キムチが刺さる)
- 近所の子 → 甘味(べったら漬け等)
- 若奥さん → さっぱり(浅漬け)

熟成で味が変わるので、古漬けにして酸味を出す、浅漬けでさっぱり売る、など戦略が広がります。
